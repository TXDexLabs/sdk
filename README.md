# TXDex

**TXDex** implements a strict **deflationary protocol** designed to strengthen the token economy over time.
Every sell transaction automatically triggers a **token burn**, continuously reducing supply and applying upward pressure on the **supply-demand ratio**.

## 🚀 Overview

TXDex is a decentralized protocol focused on creating a **self-reinforcing token economy** through automated deflation mechanics.
With each sell transaction, a portion of tokens is permanently removed from circulation, making the remaining supply more scarce.

This mechanism helps align incentives between holders, traders, and long-term ecosystem participants.

---

## ✨ Key Features

* 🔥 **Automatic Burn Mechanism**
  Every sell transaction burns a portion of tokens.

* 📉 **Deflationary Supply Model**
  The circulating supply continuously decreases.

* ⚖️ **Supply-Demand Optimization**
  Reduced supply can strengthen price dynamics over time.

* 🔗 **Decentralized Infrastructure**
  Built for transparent and trustless token economics.

* ⚡ **Efficient Transaction Processing**
  Lightweight protocol designed for speed and reliability.

## 🧠 How It Works

1. A user performs a **sell transaction**.
2. The protocol automatically calculates the **burn amount**.
3. Tokens are **permanently removed** from the total supply.
4. The reduced supply increases the scarcity of the token.

This continuous cycle supports a **long-term deflationary ecosystem**.

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/txdex.git
```

Enter the project directory:

```
cd txdex
```

Install dependencies:

```
npm install
```

---

## ▶️ Run Locally

Start a local blockchain environment:

```
npx hardhat node
```

Deploy the contract:

```
npx hardhat run scripts/deploy.js --network localhost
```
